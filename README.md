# FridgeMaster
## FridgeMaster는 나의 냉장고를 200% 활용할 수 있도록 도와주는 사이트입니다.

### I. 재료를 한눈에
- 냉장고에 남은 식품을 깔끔한 목록으로 정리합니다.
- 종류, 유통기한, 보관 위치까지 색을 통해 시각적으로 보여줍니다.

### II. AI 메뉴 추천
- 현재 냉장고에 남은 재료들을 바탕으로 AI가 맞춤 메뉴를 추천해줍니다.
- 단순 메뉴 추천에서 그치지 않고 남은 재료와의 매칭률, 조리 시간 및 방법까지 상세하게 분석해 보여줍니다.

### III. 사용자가 원하는 대로 커스터마이징
- 전체 UI를 라이트 모드 / 다크 모드로 변경 가능합니다.
- 원하는 요리 없다면 '나만의 레시피'를 직접 추가해 리스트에 추가할 수 있습니다.
